# tiny-shell

A tiny shell implementation in cpp.

## Build

```bash
$ mkdir -p build && cd build && cmake .. && cmake --build . && cd ..
```

## Run

```bash
$ ./build/tiny-shell
```

## Features

Built-in commands:
- [x] `exit`
- [x] `echo`
- [x] `type`
- [x] `pwd`
- [ ] `cd`
